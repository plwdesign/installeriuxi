PRIMEIRO PASSOS:

Execute os comandos para atualizar os pacotes do sistema. 

```
sudo apt update
```

```
sudo apt upgrade -y
```

PROXIMO COMANDO:

```bash
sudo apt install -y git && git clone https://github.com/plwdesign/installeriuxi && sudo chmod -R 777 installeriuxi && cd installeriuxi && sudo ./install_primaria
```

ACESSANDO DIRETORIO DO INSTALADOR & INICIANDO INSTALAÇÕES ADICIONAIS (USAR ESTE COMANDO PARA SEGUNDA OU MAIS INSTALAÇÃO:
```bash
cd ./instalador-iuxi && sudo ./install_instancia
```
